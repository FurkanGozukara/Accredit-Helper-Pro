    Invalid Scores Detection

    This utility will scan all exams in the database and identify students who have question scores that exceed the maximum possible score for those questions.

    What this tool does:
    • Scans all exams for students with invalid scores (scores > question max_score)
    • Calculates the student's current total score for each affected exam
    • Redistributes the total score proportionally across all questions based on each question's maximum score
    • Ensures no individual question score exceeds its maximum value
    • Creates a detailed log file showing all changes made
    Note: This operation will only affect students who currently have scores exceeding question maximums. All other scores will remain unchanged.
